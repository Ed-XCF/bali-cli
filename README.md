# bali-cli
CLI tools to simplify gRPC services and clients

### CONTRIBUTE

**Developer Environment**

```bash
make init
``` 

```examples
# setup examples environments
cd examples && git clone https://github.com/JoshYuJump/bali-cli-example-proto
```


**Tag a new release**

tag a version:

```bash
git tag -a v0.1.0
```

push tags to remote:

```bash
git push --tags
```

**Get a test report**
```bash
make test
```
